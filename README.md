# Tic Tac Toe Game

A modern Tic Tac Toe game built using React, Vite, and Tailwind CSS.

## 🚀 Live Demo

https://tic-tac-toe-game-self-theta.vercel.app

## 🛠️ Technologies Used

- React
- Vite
- Tailwind CSS
- JavaScript

## ✨ Features

- Two-player Tic Tac Toe
- Winner detection
- Restart game
- Responsive UI

## 📦 Installation

```bash
git clone https://github.com/harishthecodertobe/TicTacToeGame.git
cd TicTacToeGame/tictactoe
npm install
npm run dev
```
